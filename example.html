Game Setup

This example will cover a basic 3-player game.

To setup the game, 12 cards will be passed out to each player.
P1(Player 1) gets a hand of 2, 10, 2, 6, 4, 4, J, 5, K, 9, 10, and 5.
P2(Player 2) gets a hand of Q, 7, 6, 3, 8, 10, 6, 6, K, K, J, and 7.
P3(Player 3) gets a hand of 9, 9, J, 7, 9, 8, 8, A, Q, 4, 2, and 3.
Suits have not been included as they are not important in the game.

Gameplay

Side note: Because people usually don't like reading a lot of text, only the first 3 rounds have been shown. The rest are done after.

Round 1

P1

P1 takes the K because it is a high card.
P1 passes the remaining hand: 2, 10, 2, 6, 4, 4, J, 5, 9, 10, and 5 to P2.

P2

P2 also takes the K for the same reason as P1.
P2 passes the remaining hand: Q, 7, 6, 3, 8, 10, 6, 6, K, J, and 7 to P3.

P3

P3 takes the Q for the same reason as P1 and P2.
P3 passes the remaining hand: 9, 9, J, 7, 9, 8, 8, A, 4, 2, and 3 to P1.

Round 2

P1

P1 takes the A because P1 is hoping there is a second A.
P1 passes the remaining hand: 9, 9, J, 7, 9, 8, 8, 4, 2, and 3 to P2.
P1 now has a K and A.

P2

P2 also takes the 2 for the same reason as P1.
P2 passes the remaining hand: 10, 2, 6, 4, 4, J, 5, 9, 10, and 5 to P3.
P2 now has a K and 2.

P3

P3 takes the Q for the same reason as in round 1, hoping that the 3rd Q comes back around.
P3 passes the remaining hand: Q, 7, 6, 3, 8, 10, 6, 6, K, J, and 7 to P1.
P3 now has a Q and Q.

Round 3

P1

P1 realizes there isn't a second A, and instead takes the Q.
P1 passes the remaining hand: 7, 6, 3, 8, 10, 6, 6, K, J, and 7 to P2.
P1 now has a Q, K, and A.

P2

P2 takes the 2 because it is the second of a low number.
P2 passes the remaining hand: 9, 9, J, 7, 9, 8, 8, 4, and 3 to P3.
P2 now has a K, 2, and 2.

P3

P3 realizes there isn't a thrid Q, and takes the J instead.
P3 passes the remaining hand: 10, 2, 6, 4, 4, 5, 9, 10, and 5 to P1.
P3 now has a J, Q, and Q.

Scoring

At the end of the game, the players are left with these cards: (Sorted for your sanity)
P1: A, 3, 4, 4, 6, 9, 9, 9, 10, J, Q, K
P2: 2, 2, 2, 3, 4, 6, 7, 8, 8, 10, J, K
P3: 5, 5, 6, 6, 7, 8, 9, 10, J, Q, Q, K
